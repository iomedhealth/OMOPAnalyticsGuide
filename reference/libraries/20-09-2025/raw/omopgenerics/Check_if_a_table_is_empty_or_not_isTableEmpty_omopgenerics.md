# Check if a table is empty or not

Source: [`R/utilities.R`](https://github.com/darwin-eu/omopgenerics/blob/v1.3.1/R/utilities.R)

`isTableEmpty.Rd`

Check if a table is empty or not

## Usage
    
    
    isTableEmpty(table)

## Arguments

table
    

a table

## Value

Boolean to indicate if a cdm_table is empty (TRUE or FALSE).
