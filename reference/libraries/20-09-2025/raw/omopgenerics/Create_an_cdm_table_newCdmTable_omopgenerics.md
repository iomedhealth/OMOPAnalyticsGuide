# Create an cdm table.

Source: [`R/classCdmTable.R`](https://github.com/darwin-eu/omopgenerics/blob/v1.3.1/R/classCdmTable.R)

`newCdmTable.Rd`

Create an cdm table.

## Usage
    
    
    newCdmTable(table, src, name)

## Arguments

table
    

A table that is part of a cdm.

src
    

The source of the table.

name
    

The name of the table.

## Value

A cdm_table object
